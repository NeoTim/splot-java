Class Overview

Class for specifying additional context for operations on properties.

Instances of this class are generally passed as a variable list of arguments to property-related functions on a FunctionalEndpoint, like setProperty(PropertyKey, T, Modifier...) or fetchSection(Section, Modifier...). They provide a mechanism for indicating things like the intent to read the transition target value rather than the immediate value, or to indicate that a given change should transition to the target value over a period of time.
