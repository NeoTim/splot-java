Public Methods

public final String getName ()

Returns the name of this property, in the form <SECTION>/<TRAIT-ID>/<PROP-ID>.

  • SECTION: either SECTION_STATE, SECTION_CONFIG, or SECTION_METADATA
  • TRAIT-ID: the short identifier of the trait that this property belongs to
  • PROP-ID: the short identifier of the property
