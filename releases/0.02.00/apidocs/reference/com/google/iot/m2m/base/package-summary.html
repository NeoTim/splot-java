package

com.google.iot.m2m.base

Splot base classes and interfaces.

Interfaces

ChildListener Interface for handling changes to a FunctionalEndpoint's children. 
FunctionalEndpoint Interface for a Functional Endpoint. 
Group Interface for managing a group. 
LocalTrait Interface which is implemented by helper classes (like BaseTrait.AbstractLocalTrait) which are generated exclusively by the trait annotation processor. 
PersistentStateInterface Interface for enabling the persistent storage of information related to an object. 
PersistentStateListener Interface used to receive updates from a PersistentStateInterface regarding the need to refresh the data stored in non-volatile memory. 
PersistentStateManager A persistent state manager handles the retrieval and updating of the persistent state of objects which implement PersistentStateInterface. 
PropertyListener<T> Interface for receiving asynchronous notifications for changes to individual properties on a FunctionalEndpoint. 
SectionListener Interface for receiving notifications of changes to the properties of a Section in a FunctionalEndpoint. 
Technology Interface for discovering, interacting with, and hosting FunctionalEndpoints. 

Classes

DiscoveryBuilder Builder class for constructing a new DiscoveryQuery object. 
DiscoveryQuery An abstract class representing a specific functional endpoint discovery query. 
DiscoveryQuery.Listener Callback class for handling functional endpoints immediately as they are discovered and being notified of the completion of a DiscoveryQuery. 
MethodKey<T> A generic class for identifying and facilitating return-type-safety for methods. 
Modifier Class for specifying additional context for operations on properties. 
Modifier.All Modifier class indicating that properties with null values should be included in the section results. 
Modifier.Duration Modifier class indicating a transition duration for changes to properties in STATE. 
Modifier.Increment Mutation class that indicates that the property value should be incremented. 
Modifier.Insert Mutation class that indicates that the value should be inserted into the property. 
Modifier.Mutation Mutation superclass for modifiers that indicate the property value should be mutated. 
Modifier.Remove Mutation class that indicates that the value should be removed from the property. 
Modifier.Toggle Mutation class that indicates that the property value should be toggled. 
Modifier.TransitionTarget Modifier class indicating that the transition target value is desired, rather than the immediate value. 
ParamKey<T> A generic class for identifying and facilitating improved type-safety for method parameters. 
PropertyKey<T> A generic class for facilitating type-safety on individual properties while remaining convenient to use. 
Splot Splot-related constants and utility functions. 
TypeConverter<T> Class for performing Splot's implicit value conversions. 
TypedKey<T> Contains the shared implementation of PropertyKey, MethodKey, and ParamKey. 
TypedKeyValue<T> Convenience class representing a value tagged with a TypedKey.
