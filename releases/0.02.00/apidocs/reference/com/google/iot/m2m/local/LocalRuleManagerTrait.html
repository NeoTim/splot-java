Class Overview

Manager trait for enabling the in-band creation and management of LocalRule instances. This trait is implemented by the LocalAutomationManager, but is publicly defined to allow it to be integrated into custom LocalFunctionalEndpoint implementations.

public Map<String, Object> copyPersistentState ()

Retrieves a copy of the state for the instance that should be saved to nonvolatile storage. When reconstructing the associated instance, call initWithPersistentState(Map) to restore this state.

The data contained in the returned Map should be considered opaque unless specified otherwise by the implementing technology.

The types for the values contained in the map are only guaranteed to be faithfully restoreable if they are one of the following types:

  • Integer
  • Double
  • Float
  • byte[]
  • String
  • Map<String,Object>, where Object is also one of these acceptable types
  • java.util.List<Object>, where Object is also one of these acceptable types
This allows the object to be easily serialized to CBOR.

This method MUST be thread safe.

Returns
  • the persistent state of this instance

public void initWithPersistentState (Map<String, Object> persistentState)

Restores a persistent state previously retrieved from copyPersistentState(). This should be performed only once when this instance is being initialized. The behavior for calling this method more than once is undefined.

Parameters
persistentState a map previously returned from copyPersistentState(). May be null if there is not yet any persistent state associated with this object.
