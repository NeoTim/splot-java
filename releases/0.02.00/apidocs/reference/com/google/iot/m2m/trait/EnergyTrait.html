Class Overview

The Energy trait contains properties that relate to the energy consumption of a device. A FE would rarely implement all of the described properties: only the relevant properties would be implemented.

public static final PropertyKey<Float> CONF_MAX_AMPS

The maximum number of amps that the load is allowed to draw before the FE automatically turns off the load. Set to null to disable. When tripped, STAT_TRAP is set to "tag:google.com,2018:m2m:traits:energy:exception#max-amps" until the condition is reset by turning the load on again.

public static final PropertyKey<Float> CONF_MAX_VOLTS

The voltage above which the load will be automatically turned off by the FE. Set to null to disable. When tripped, STAT_TRAP is set to "tag:google.com,2018:m2m:traits:energy:exception#max-volts" until the condition is reset by turning the load on again.

public static final PropertyKey<Float> CONF_MAX_VOLT_AMPS

The maximum apparent power (volt-amps) that the load is allowed to draw before the FE automatically turns off the load. Set to null to disable. This property requires that the OnOff trait also be supported. When tripped, STAT_TRAP is set to "tag:google.com,2018:m2m:traits:energy:exception#max-volt-amps" until the condition is reset by turning the load on again.

public static final PropertyKey<Float> CONF_MAX_WATTS

The maximum power that the load is allowed to draw before the FE automatically turns off the load. Set to null to disable. When tripped, STAT_TRAP is set to "tag:google.com,2018:m2m:traits:energy:exception#max-watts" until the condition is reset by turning the load on again.

public static final PropertyKey<Float> CONF_MIN_VOLTS

The voltage below which the load will be automatically turned off by the FE. Set to null to disable. When tripped, STAT_TRAP is set to "tag:google.com,2018:m2m:traits:energy:exception#min-volts" until the condition is reset by turning the load on again.

public static final PropertyKey<Long> STAT_ENERGY

The accumulated power (energy) used over time by the FE, measured in watt-hours. If this functional endpoint allows this value to be reset, it can be reset by setting its value to zero or null. Setting to any other value MUST fail.

public static final PropertyKey<Float> STAT_POWER_FACTOR

The power factor measured by this FE. Only meaningful when measuring AC. Unitless.

public static final PropertyKey<Float> STAT_VOLTS

Instantaneous electric potential, measured in volts.

public static final PropertyKey<Float> STAT_VOLT_AMPS

Apparent instantaneous power draw, measured in volt-amps. Note that this is literally the volts multiplied by the amps, so this will differ if the power factor is anything other than 1.0. Only really meaningful when measuring AC.

public static final PropertyKey<Float> STAT_WATTS

Instantaneous power draw, measured in watts. Unlike the property volt-amps, this property takes into consideration power factor when measuring AC.
