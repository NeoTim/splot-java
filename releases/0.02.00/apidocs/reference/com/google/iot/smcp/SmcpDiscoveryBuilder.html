Class Overview

SMCP-specific subclass of DiscoveryBuilder which adds some CoAP-specific methods.

Instances of this class are created via createDiscoveryQueryBuilder().

public SmcpDiscoveryBuilder includeHosted (boolean includeHosted)

Determines if functional endpoints that are hosted by this Technology should be included in the results. Default value is false. If the technology does not support hosting, then this method does nothing.

Parameters
includeHosted true if hosted functional endpoint should be included in discovery results, false otherwise.
Returns
  • a convenience reference to this discovery builder object.

public DiscoveryBuilder mustBeGroup ()

Specifies that the discovery results should only include groups. Default is to return all discovered functional endpoints, including groups. If the technology does not support groups, this method will cause the resulting DiscoveryQuery to return nothing.

Returns
  • a convenience reference to this discovery builder object.

public SmcpDiscoveryBuilder mustHaveTrait (String traitShortName)

Indicates that this discovery operation is to be limited to returning functional endpoints which have the specified traits, as identified by the trait's short name. This method may be called multiple times to require multiple traits to be present in the discovered functional endpoints.

Parameters
traitShortName the trait short name to match on discovered functional endpoints.
Returns
  • a convenience reference to this discovery builder object.

public SmcpDiscoveryBuilder mustHaveUid (String uid)

Indicates that this discovery operation is to be limited to returning functional endpoints which have the specified UID.

If this criteria is specified, there is generally only a single FunctionalEndpoint (or no functional endpoints) returned. However, more may be returned if there is a UID conflict.

Parameters
uid the uid string to match for discovered functional endpoints.
Returns
  • a convenience reference to this discovery builder object.

public DiscoveryBuilder mustNotBeGroup ()

Specifies that the discovery results should only include groups. Default is to return all discovered functional endpoints, including groups. If the technology does not support groups, then this method does nothing.

Returns
  • a convenience reference to this discovery builder object.

public SmcpDiscoveryBuilder setMaxResults (int count)

Determines the maximum number of functional endpoints which can be discovered before the query automatically terminates. A value of zero (0) indicates that there is no explicitly defined maximum. The default value is zero.

Parameters
count The maximum number of functional endpoints to discover, or zero if there is no maximum.
Returns
  • a convenience reference to this discovery builder object.

public SmcpDiscoveryBuilder setTimeout (long timeout, TimeUnit units)

Determines the maximum amount of time that should be spent discovering functional endpoints. The default value is technology-specific, but generally not longer than 10 seconds.

Parameters
timeout the duration of the discovery operation, measured in units.
units the units of timeout.
Returns
  • a convenience reference to this discovery builder object.
